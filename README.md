# Piano Technique Analysis System

**Project in progress**

## Overview
A wearable system that analyzes piano technique using sensor data from a glove. The system captures how notes are played by measuring finger force, hand posture, motion, and sound timing.

## Features
- Finger force sensing (FSR)
- Hand posture tracking
- Motion tracking (IMU)
- Audio-based note onset detection
- Timing and consistency analysis
- Real-time visualization and feedback

## System Architecture
Glove Sensors → External Module → Laptop → Processing + Visualization

## Purpose
This project aims to make piano technique measurable by providing objective feedback on timing, consistency, and posture, areas that are typically evaluated subjectively.
